# TypeScript RPG Arena

Um RPG de arena em TypeScript, jogável no navegador.

## Como rodar o projeto (terminal)

1. Instale as dependências:
   ```sh
   npm install
   ```
2. Compile o TypeScript:
   ```sh
   npm run build
   ```
3. Rode o jogo no terminal:
   ```sh
   npm start
   ```

3. Acesse `http://127.0.0.1:PORT` no navegador.
**OBS:** A porta será mostrada no terminal, caso o navegador não abra por padrão, utilize a porta que foi exibida.

## Como jogar
- Escolha seu personagem e um adversário.
- A cada turno, escolha uma ação:
  - Atacar
  - Usar habilidade (se disponível)
  - Usar item (se disponível)
- O objetivo é derrotar o adversário em combate por turnos.
- O jogo mostra o status, inventário e habilidades a cada rodada.
- No navegador, use o campo de texto e pressione Enter ou clique em "Enviar" para jogar. Use o botão "Reiniciar Jogo" para começar de novo.
